# TestGemini
Tienes que instalar primero uvicorn y meterlo en el path: https://uvicorn.dev/ 
Lo puedes instalar con   
```pip uvicorn```  
 y comprobar que la carpeta ```C:\Python310\Scripts\``` está en las variables de entorno, el path.

Para que funcione GEMINI debes incluir la APIKEY en el fichero Gemini_APIKEY.txt dentro de la carpeta PythonServer, la APIKEY se encuentra en el notion en la pestaña de gemini.
